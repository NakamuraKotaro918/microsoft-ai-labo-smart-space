# 機能仕様書：画面表示項目とデータ取得元

**作成日:** 2025年8月6日  
**バージョン:** 1.1  
**対象システム:** Microsoft AI Labo スマート空間最適化ダッシュボード

## システム概要

本システムは表示専用のダッシュボードシステムです。ユーザー操作は一切行わず、以下の6つの画面を10秒間隔で自動切り替えして表示します：

1. **エントランス基本情報画面** - 来場者数と基本システム情報
2. **エントランス属性分析画面** - 年齢層・性別分布の詳細分析
3. **エントランス時間別分析画面** - 時間別来場者数とピーク分析
4. **エントランス行動パターン分析画面** - 来場者の行動パターン一覧と割合
5. **個人開発部屋環境データ画面** - 温湿度・CO2・利用状況
6. **個人開発部屋トレンド分析画面** - 環境データの24時間トレンド

各画面は1画面内に全情報を収め、スクロール不要で表示されます。画面倍率100%で見切れることなく表示されるよう最適化されています。

## 7. 自動表示システム仕様

### 3.1 画面切り替え

| 項目 | 仕様 | 説明 |
|------|------|------|
| 切り替え間隔 | 10秒 | 各画面を10秒間表示後、次の画面に自動切り替え |
| 切り替え順序 | エントランス基本情報 → エントランス属性分析 → エントランス時間別分析 → エントランス行動パターン分析 → 個人開発部屋環境データ → 個人開発部屋トレンド分析 → エントランス基本情報... | 6画面を循環表示 |
| 画面表示方式 | 1画面完結 | 各画面はスクロール不要で全情報を表示 |
| ユーザー操作 | なし | 完全自動表示、操作ボタン・制御機能なし |

### 3.2 表示インジケーター

| 表示項目 | 説明 |
|---------|------|
| 現在画面名 | 表示中の画面名を表示 |
| 自動切替インジケーター | 自動切り替え動作中を示すアニメーション |
| 切り替えカウントダウン | 次の画面切り替えまでの残り秒数（10→9→8→...→2→1→10...） |
| 画面進捗表示 | 現在の画面番号/総画面数（例：4/6） |

#### カウントダウン表示仕様
- **表示形式:** 数字（1-10）
- **更新頻度:** 1秒間隔
- **色変化:** 
  - 10-6秒: 緑色（#4CAF50）
  - 5-4秒: オレンジ色（#FF9800）
  - 3-1秒: 赤色（#FF5722）
- **アニメーション:** パルス効果（拡大縮小）

## 7. データ精度・信頼性

### 6.1 人物認識カメラ

- **年齢推定精度:** ±5歳程度
- **性別推定精度:** 95%以上
- **人物検出精度:** 98%以上（良好な照明条件下）

### 6.2 AI分析システム

- **行動分析精度:** 90%以上
- **感情推定精度:** 85%以上
- **特徴抽出精度:** 88%以上

### 6.3 快適君センサー

- **温度測定精度:** ±0.5°C
- **湿度測定精度:** ±3%RH
- **CO2測定精度:** ±50ppm

## 9. 注意事項

1. **表示専用システム:** 本システムは表示専用で、ユーザー操作・制御機能は一切ありません
2. **自動切り替え:** 画面は10秒間隔で自動切り替えされ、手動操作はできません（6画面循環）
3. **100%倍率対応:** 画面倍率100%で見切れることなく表示されるよう最適化されています
4. **スクロール不要設計:** 各画面は1画面内に全情報を収め、スクロール操作は不要です
5. **システム稼働状況表示なし:** システムの詳細な稼働状況は表示せず、必要最小限の情報のみ表示します
6. **プライバシー保護:** 人物画像データは分析後即座に削除され、個人を特定できる情報は保存されません
7. **データ保持期間:** 統計データは30日間、環境データは90日間保持されます
8. **システム依存性:** 各センサーの稼働状況により、一部データが取得できない場合があります
9. **精度変動:** 環境条件（照明、天候等）により、検出・分析精度が変動する場合があります
10. **画面進捗表示:** 現在表示中の画面番号と総画面数が常時表示されます
